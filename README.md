Deskripsi  project: Aplikasi Aceh Kuliner ini merupakan aplikasi yang berisi berbagai macam makanan khas aceh, 
dimana aplikasi ini menyediakan berbagai macam resep makanan khas aceh dan juga proses pembuatannya. 
Tujuan dari aplikasi ini adalah untuk memberikan informasi kepada masyarakat luas, khususnya luar aceh agar bisa membuat makanan khas aceh.
